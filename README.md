## Docker Compose
>**Run postgres in a docker container**  
`docker-compose up`  
`docker-compose up db`

>**Exec into postgres container**  
`docker exec -it jamesvrooney-postgres /bin/sh`

>**Use postgres cli (within the container):**  
`psql -U postgres`
